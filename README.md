# Better Party Finder (KR Custom)

파이널 판타지 14의 파티 찾기에 고급 필터 프리셋을 제공하는 [Dalamud](https://github.com/goatcorp/Dalamud) 플러그인입니다.

> 이 저장소는 [Infiziert90/BetterPartyFinder](https://github.com/Infiziert90/BetterPartyFinder)를 **한국 서버용으로 수정한 포크 빌드**입니다.

## 주요 기능

- 필터 프리셋을 만들어 손쉽게 전환
- 임무/카테고리 화이트리스트·블랙리스트
- 특정 플레이어의 파티 모집 숨기기
- 아이템 레벨 범위 필터, 복수 잡·슬롯 필터 (예: MCH + GNB 자리)

### 한국 서버용 추가 기능

- 한국어 UI
- 한국 파티 찾기 용어 기반 **빠른 키워드 필터링** — 2상자/파밍, 상자 무관, 헤딩, 초행, 클목/성불, 연습, 갱신, N클, 조기 해산, 클각 시 자살, 출발 시각 지정 등
- 직업 필터링 확장 — 확정 직업 예약 판정, 직업 중복 없음

## 설치

1. `/xlsettings` → **Experimental**(실험적) 탭의 **Custom Plugin Repositories**에 아래 주소를 추가합니다:
   ```
   https://raw.githubusercontent.com/pnsdh/DalamudPlugins/main/pluginmaster.json
   ```
2. 플러그인 설치 창(`/xlplugins`)에서 **Better Party Finder (KR Custom)**을 설치합니다.

## 명령어

| 명령어 | 설명 |
|---|---|
| `/betterpartyfinder` (또는 `/bpf`) | 메인 인터페이스 열기. `c` 또는 `config` 인자로 설정 창 열기 |
